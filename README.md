# WechatChatBot

此程序由Python3编写，只实现了自动回复功能

# 使用方法
- pip3 install requirement.txt
- python3 chatbot.py
- 二维码会直接显示在Terminal
- 手机扫码登陆

# 服务器后台使用:
- pip3 install requirement.txt
- nohup python3 chatbot.py &
- sudo cat nohup.txt
- 手机扫码登陆

实测在服务器可以运行至少48小时以上

*注: 代码借鉴于 [WeixinBot](https://github.com/Urinx/WeixinBot)
